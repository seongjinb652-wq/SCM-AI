\# SCM-AI: 재고 관리 최적화



\## 프로젝트 개요

공급망(Supply Chain)에서 재고 관리 최적화를 강화학습(RL)과 정책 기반 접근으로 해결



\## 핵심 목표

...



\## 구체적 방향

1\. 전통적 정책 구현: EOQ(경제적 주문량), (s, S) 정책 등 기본 재고 정책을 코드로 구현

2\. 강화학습 적용: Gym, PyTorch 기반 RL 알고리즘을 활용해 동적 환경에서 최적 정책 학습

3\. 시뮬레이션 평가: 비용 최소화와 서비스 수준 유지라는 두 가지 목표를 동시에 달성

4\. 비교·검증: 전통적 정책 vs RL 기반 최적화 성능을 비교하고 검증



\## 실험 계획



\### 데이터 현황

\- \*\*현재\*\*: 20 품목, 2년치 데이터 (2024~2025)

\- \*\*향후\*\*: 5,000 품목 확장 예정



\### 데이터 분할 전략

\- \*\*옵션 1\*\*: 학습 18개월 / 검증 6개월

\- \*\*옵션 2\*\*: 학습 21개월 / 검증 3개월

\- \*\*결정 기준\*\*: 초기 실험 후 성능 평가하여 확정



\### 비교 시나리오



\#### 시나리오 1: 기존 운영 방식

\- 2025년 7월 이전 운영 방식 그대로 유지



\#### 시나리오 2: 파라미터 고정 방식

\- 2025년 7월에 파라미터 1회 조정

\- 이후 고정 운영



\#### 시나리오 3: RL 적응형 방식 (제안)

\- 매월 강화학습으로 파라미터 재조정

\- 수요 변동에 동적 대응



\### 성능 평가 지표

\- 입고 횟수 (정기 입고 vs 긴급 입고)

\- 출고 횟수

\- 입고 수량

\- 재고 유지 비용

\- 서비스 수준 (품절률)



\### 하이퍼파라미터 튜닝 전략

학습 속도를 위해 점진적 세분화 방식 적용:



1\. \*\*분기별 집계\*\* → 빠른 프로토타입 (초기 실험)

2\. \*\*월별 집계\*\* → 중간 정밀도 (파라미터 탐색)

3\. \*\*주별 집계\*\* → 높은 정밀도 (세부 튜닝)

4\. \*\*일별 집계\*\* → 최종 검증 (실전 배포)

\## 재고 최적화 알고리즘 설계 (안)



> \*\*Note\*\*: 현재 개발 방향이며, 실험 결과에 따라 조정될 수 있음



\### 전체 흐름

스파이크 제거 → 안정적 통계 → Min-Max/최저 발주량 규칙 → 변동성 대응 → 운영 목표 달성



---



\### 1. 스파이크 제거 및 통계 산출



\*\*목적\*\*: 급증 구간을 제외하고 안정적인 수요 패턴 파악



\*\*방법\*\*:

\- 일별 수요 데이터에서 \*\*상위 10~20% 제외\*\* (트리밍)

\- 나머지 데이터로 평균, 중앙값, MAD(중앙값 절대편차) 계산

\- 변동성 과대평가 방지



---



\### 2. Min-Max 및 최저 발주량 규칙



\*\*Min (재주문점)\*\*:

```

Min = 리드타임 동안 평균 수요 + 안전재고

안전재고 = MAD × 안전계수(1.5)

```



\*\*Max (목표 재고 수준)\*\*:

```

Max = Min + 추가 1~2주치 평균 수요

```



\*\*최저 발주량\*\*:

```

최저 발주량 = 최근 2~4주 평균 수요 × 0.6~0.8

목적: 소량 발주 방지, 발주 효율성 확보

```



---



\### 3. 스파이크 대응 규칙



\*\*스파이크 정의\*\*: 상위 10~20% 급증 구간



\*\*대응 전략\*\*:

\- \*\*즉시 대응\*\*: 긴급 발주 (상한선 설정)

\- \*\*사후 조정\*\*: 스파이크 이후 3~5일간 일반 발주량 감소

\- \*\*과잉 재고 방지\*\*: 긴급 발주량 상한선 적용



---



\### 4. 변동성 대응 고도화



\*\*추세 반영\*\*:

\- 중앙값 대신 \*\*EMA(지수이동평균)\*\* 활용

\- 최근 데이터에 더 높은 가중치



\*\*리드타임 기반 조정\*\*:

\- 짧은 리드타임 → Max 작게 설정

\- 긴 리드타임 → Max 크게 설정



\*\*품목별 차별화\*\*:

\- 중요 품목 → 높은 서비스 수준(z값 ↑)

\- 일반 품목 → 표준 서비스 수준



---



\### 5. 초기 구현 기준



| 파라미터 | 초기값 | 비고 |

|---------|--------|------|

| 트리밍 비율 | 상위 15% | 스파이크 제거 |

| 베이스 수요 | 트리밍 후 중앙값 | 안정적 기준 |

| 안전계수 | MAD × 1.5 | 변동 여유 |

| Min | 베이스 × 리드타임 + 안전재고 | 재주문점 |

| Max | Min + 베이스 × 7일 | 목표 재고 |

| 최저 발주량 | 14일 평균 × 0.7 | 발주 효율 |



---



\### 6. 운영 목표 (KPI)



| 지표 | 현재 | 목표 | 개선 방향 |

|-----|------|------|----------|

| 계획 발주 비율 | ~50% | 70~85% | ↑ |

| 긴급 발주 비율 | ~50% | 25~30% | ↓ |

| 재고 회전율 | - | 유지 또는 개선 | → |

| 품절률 | - | 5% 이하 | ↓ |



\*\*핵심 전략\*\*: 

\- 재고를 크게 늘리지 않으면서

\- 안정적인 계획 발주 중심 운영 달성
